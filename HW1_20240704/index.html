เกี่ยวกับฉัน
สวัสดีครับผมชื่อณัฐ ในช่วง Covid-19 ผมได้รับมอบหมายให้ทำระบบลงทะเบียน โดยผมอยากให้ผู้ลงสามารถค้นหาข้อมูลของตัวเองได้ เพื่อที่จะได้ไม่ต้องกรอกข้อมูลของตัวเองใหม่ทั้งหมด ผมจึงได้เริ่มต้นศึกษาวิธีการทำจากอินเตอร์เน็ต จนสามารถทำได้สำเร็จ และงานในครั้งนี้ทำให้ผมรู้สึกชอบในการเขียนเว็บไซต์ ผมจึงได้ศึกษาหาความรู้เพื่อต่อยอดจนมาตลอด โดยในปัจจุบันผมสามารถเขียนเว็บไซต์เพื่อทำระบบต่าง ๆ ได้ เช่น ระบบลงทะเบียนโดยใช้ QR Code โดยใช้ HTML CSS JS และ Google Script + Google Sheet หรือ Firebase (RTDB & auth)
